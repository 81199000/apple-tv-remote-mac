# 参与贡献

感谢参与“遥控器助手”开源项目。

## 开发环境

1. 安装 Xcode Command Line Tools：`xcode-select --install`
2. 执行 `./build.sh` 编译，再执行 `./create_app_bundle.sh` 创建应用程序包。
3. 使用真实的第一代 Apple TV 遥控器测试，并授予辅助功能权限。

## 提交代码

- 使用清晰的中文说明描述问题和修改内容。
- 保持 Swift 代码风格一致，优先使用安全、可维护的实现。
- 提交前至少运行一次 `./build.sh`。
- Pull Request 请说明测试的 macOS 版本和遥控器型号。
